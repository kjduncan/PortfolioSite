SideDoor Inc.

Tech Stack: Vagrant, Gulp, HTML, CSS, LESS, Bootstrap, jQuery, PHP, Photoshop, Illustrator

SideDoor Inc. is a startup in Boulder, Colorado that works to help people buy and sell homes without real estate agents. I currently work as a User Experience Designer and Frontend Developer. Since I started, my job has been to help with the redesign of SideDoor's web based platform, focusing on usability, functionality, and over all experience. I focus primarily on the design and layout of pages as well as the frontend code. For design, I use Illustrator and Photoshop to create mockups for the desktop, tablet, and mobile versions of the site. I then send those designs for review, tweak what needs tweaking, and begin coding. This has been my first major project using Bootstrap and LESS which has been a great challenge for me. I have loved becoming more familiar with these technologies and appreciate how they can be utilized to simplify code and save time. It has also been a great opportunity and creative challenge to head up the design for this project. The site is now complete and will be launching in the next few weeks.
