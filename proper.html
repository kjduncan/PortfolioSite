Proper Digital

Tech Stack: Jekyll, HTML, CSS, SASS, jQuery, AWS

Laying things out in Jekyll was a little different than what I was used to, but in the end, I appreciated the efficiency and organization it brought to the code base. I also incorperated a drop down nav bar using jquery which was an interesting new challenge for me.
