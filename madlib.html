Madlib Game

This is a personal project that I worked on to practice jquery. I decided to expand on a madlib project I created in General Assembly's Dash where users would be asked for specific input and then that input would be compiled into a short story. I improved accessibility features by adding keyboard shortcuts and error handling. I really enjoyed problem solving throughout this project, trying different strategies with jquery to make things work.
