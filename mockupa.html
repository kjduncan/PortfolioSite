Mockup-A Weblab

This is another project I worked on in the HTML portion of Weblab Development Bootcamp. We were required to code a pixel perfect webpage from a Zeplin mockup using HTML and CSS. This was the first time that I had worked from a mockup with a lot of different components and it was difficult at first to figure out the positioning of each element on the page and make everything cohesive. I really enjoyed working through the challenges presented in this project and learned a ton about layout in HTML.
