Daily UI

Tech Stack: Photoshop, Illustrator

Daily UI has been a fun creative challenge for me. They send a design prompt email everyday for 100 days. These are a few of the things I have come up with based on Daily UI prompts. I typically use Photoshop and Illustrator. Something I try to focus on is trying to solve these problems in a more creative way, adding little features that I think improve the overall experience. For example, on the calculator, I added the ability to see the calculations as well as the total and to tap and edit without having to delete everything and start over. On the sign in form, I tried to simplify and only ask the user for necessary information. I also removed the confirm email field to reduce friction.
